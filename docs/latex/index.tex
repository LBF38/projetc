The C project consists in loading data into a list using two methods.\hypertarget{index_autotoc_md2}{}\doxysubsection{Method 1}\label{index_autotoc_md2}
It is a simple list. We compare the cells made from lines and then insert the cell in the right place.\hypertarget{index_autotoc_md3}{}\doxysubsection{Method 2}\label{index_autotoc_md3}
It is an optimized list using an index. We compare the cells made from lines with an index and then insert the cell in the right index\+\_\+list.\hypertarget{index_autotoc_md4}{}\doxysubsection{Results}\label{index_autotoc_md4}
We compare the execution time of both methods using small, medium and large files.
