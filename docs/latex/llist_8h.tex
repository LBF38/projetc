It compares the cell we want to insert with the index letters and then insert it in the right place.

There are 4 cases \+: ~\newline

\begin{DoxyItemize}

\item the list doesn\textquotesingle{}t have the lcell at the beginning. We create and insert a lcell at the top of the llist.
\item the list doesn\textquotesingle{}t have the lcell at the end. We create and insert a lcell at the end of the llist.
\item the list has the lcell and we insert the cell in the index\+\_\+list.
